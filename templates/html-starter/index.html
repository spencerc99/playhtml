playhtml getting started

This is the playhtml starter. You can use it to build your own tiny social network. Check out the code and hover over the capabilities below to see which of these things are collaborative elements.

  1. can-play (custom capabilities with your code!)
  2. can-move
  3. can-mirror
  4. can-toggle
  5. can-duplicate
  6. can-spin
  7. can-grow
  8. can-duplicate
  9. can-hover

For all the details about these capabilities and more, see the playhtml docs
