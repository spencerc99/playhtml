# playhtml Templates

Quick-start templates to get you building with playhtml.

## HTML Starter

A minimal HTML template showcasing playhtml capabilities with vanilla JavaScript.

```bash
npx degit playhtml/playhtml/templates/html-starter my-project
open my-project/index.html
```

Then open `index.html` in your browser.

## React Starter

A React + Vite template with playhtml components.

```bash
npx degit playhtml/playhtml/templates/react-starter my-project
cd my-project
bun install
bun dev
```

The dev server will start at http://localhost:5173
