playhtml turns any HTML element into a live, collaborative one with a single attribute. This guide gets you from empty page to a working shared toggle.

## Install

Add the package with your package manager:

```bash
npm install playhtml
```

Or load it from a CDN without a build step:

```html
<script type="module" src="https://unpkg.com/playhtml"></script>
```

## Initialize

Call `playhtml.init()` once on page load. Cursors are opt-in.

```js
import { playhtml } from "playhtml";

playhtml.init({ cursors: { enabled: true } });
```

Then mark any element with a capability attribute — for example, `can-toggle`:

```html
<button id="my-lamp" can-toggle>lamp</button>
```

## React setup

In a React app, wrap your tree in `PlayProvider` and use one of the capability components:

```tsx
import { PlayProvider, CanPlayElement } from "@playhtml/react";
import { TagType } from "@playhtml/common";

export function App() {
  return (
    <PlayProvider initOptions={{ cursors: { enabled: true } }}>
      <CanPlayElement
        tagInfo={[TagType.CanToggle]}
        id="my-lamp"
        defaultData={{ on: false }}
      >
        {({ data, setData }) => (
          <button onClick={() => setData({ on: !data.on })}>
            {data.on ? "on" : "off"}
          </button>
        )}
      </CanPlayElement>
    </PlayProvider>
  );
}
```
